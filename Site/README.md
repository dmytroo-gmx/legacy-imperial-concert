# Legacy Imperial Concerts

Premium 3D landing page for the Legacy Imperial Concerts agency (Poland).

## Features

- Cinematic video background with parallax
- Three.js 3D scene: golden particles, floating diamonds, orbital light rings
- Multi-language support: PL / UA / BY / RU / EN
- Custom cursor with magnetic buttons
- 3D tilt cards with shine effect
- Reveal-on-scroll animations
- Mobile-responsive
- Working contact forms (client-side)

## Stack

- Pure HTML/CSS/JS — no build step required
- Three.js (CDN) for 3D effects
- Google Fonts (Playfair Display + DM Sans)

## Run locally

```bash
python -m http.server 8765
# open http://localhost:8765/
```

## Deploy

Any static host works: Vercel, Netlify, GitHub Pages, Cloudflare Pages.
